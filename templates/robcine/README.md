# Robcine Template

A cinematography/photography portfolio template with a dark theme, gallery with lightbox, blog, rental page, and hire me section.

## Features

- **Gallery** — Grid of images with full-screen lightbox
- **Blog** — Expandable blog posts loaded from database
- **About** — Bio page with behind-the-scenes image gallery
- **Rental** — Equipment rental page with pricing
- **Hire Me** — Services and contact information

## Pages

- `index.html` — Main gallery page with hero section
- `about.html` — About/bio page with BTS gallery
- `blog.html` — Blog posts listing
- `rental.html` — Equipment rental with booking form
- `hire.html` — Services and rates

## Customization

All content is loaded dynamically from the Sidey CMS database. Customize via the dashboard:

- **Site Title & Description** — Hero section text
- **About Content** — Biography text
- **Gallery Images** — Upload via dashboard
- **Blog Posts** — Create/edit in dashboard
- **Rental Items** — Configure equipment and pricing
- **Hire Details** — Set rates and services
